Abstract

This paper presents Sounz, a collaborative web-based social platform for musicians, designed to streamline the music production process through real-time interaction, audio sharing, and cloud-based project management. The system supports multi-user audio editing, copyright protection via audio fingerprinting, and integrated chat functionalities — all aimed at connecting artists globally within a secure and responsive interface.

Key Highlights

Built With: Django (Python), HTML, CSS, JavaScript, SQLite

Core Features:
• Real-time collaboration tools
• Built-in chat and notification system
• Copyright detection using MFCC & audio fingerprinting
• Cloud-based multi-audio mixing algorithm
• Responsive UI for all devices

Security: Media visibility controls and structured content reporting system

Technical Contributions

• Introduced a novel algorithm for mixing multiple audio files.
• Implemented a copyright detection algorithm using MFCC distance thresholds.
• Achieved seamless integration between content collaboration, user management, and copyright validation.
• Designed with a focus on user-friendly navigation and scalability.
